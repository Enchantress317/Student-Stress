# Student Stress Assessment & Machine Learning Analysis

## Overview

A data analytics and machine learning project focused on analyzing stress-related factors and predicting **Low, Medium, and High Stress levels** using questionnaire-based data.

## Key Analysis

* Exploratory Data Analysis and Correlation Analysis
* Cronbach's Alpha Reliability Testing
* ANOVA and Welch's T-Test
* K-Means, K-Medoids, DBSCAN, and Agglomerative Clustering
* PCA for dimensionality reduction and cluster visualization
* Feature Importance Analysis
* Machine Learning Model Comparison and Hyperparameter Tuning

## Machine Learning Models

* Random Forest
* Extra Trees
* AdaBoost
* Decision Tree
* XGBoost

The **tuned Extra Trees Classifier** achieved the best performance with **74.24% accuracy** and **72.77% weighted F1-score**.

## Key Findings

* The questionnaire demonstrated strong internal reliability with a **Cronbach's Alpha of 0.8895**.
* Living arrangement showed a statistically significant relationship with stress levels.
* K-Means clustering was used to identify **Low, Medium, and High Stress groups**.
* Extra Trees was the best-performing classification model.
